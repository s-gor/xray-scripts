\# Установка Xray xHTTP + Nginx



1\. Зарегистрировать домен в Dynu.

2\. Убедиться, что домен резолвится в IP сервера.

3\. Установить Nginx.

4\. Проверить доступность порта 80.

5\. Выпустить сертификат Let's Encrypt через acme.sh.

6\. Установить Xray.

7\. Настроить inbound xHTTP на 127.0.0.1:8443.

8\. Настроить Nginx reverse proxy для /xhttp.

9\. Проверить работу клиента.

10\. Разместить веб-страницу в /var/www/html.
