\# Xray xHTTP + Nginx



Проверенная рабочая конфигурация xHTTP за Nginx.



Используется:



\- Ubuntu 26.04

\- Xray 26.3.27

\- Nginx 1.28.3

\- Let's Encrypt (acme.sh)

\- Dynu DNS



Nginx обслуживает HTTPS на порту 443 и одновременно отдаёт веб-страницу. Запросы к пути `/xhttp` передаются на локальный экземпляр Xray, который слушает порт 8443.



Конфигурация проверена на AWS EC2.
